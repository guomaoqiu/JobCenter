# 特点:
* 可视化界面操作
* 定时任务统一管理
* 完全兼容Crontab
* 支持秒级定时任务
* 作业任务可搜索、暂停、编辑、删除
* 作业任务持久化存储、三种不同触发器类型作业动态添加

# 用法:
```
$ git clone https://github.com/guomaoqiu/JobCenter.git
$ cd JobCenter
$ pipenv install --dev
$ pipenv shell
# 需提前创建数据库
$ flask init 

$ flask run
* Running on http://127.0.0.1:5000/
```

# 常用命令

```
$ pipenv install  # 创建虚拟环境并安装依赖
$ pipenv shell  # 激活虚拟环境
$ flask initdb  # 初始化数据库
$ flask admin  # 创建管理员账户
$ flask initdb --drop # 删除数据
```
